---

# Commands

## To install:

#### `yarn install`

## To run:

#### `yarn start`

## Environment variables in `/frontend`:

You must only specify the following variable:

`REACT_APP_API_URL=API URL`

## Build & Deploy

Build: `yarn build`

Deploy: `netlify deploy --prod --dir ./build` (must specify the `./build` directory)
