# Express backend

---

## To install:

#### `yarn install`

## To run:

#### `yarn start`

## Environment variables in `/backend`:

- `PORT`=8000
- `ADMIN_USER`= [_user_]
- `ADMIN_PASSWORD`= [_password_]
- `FIREBASE_PROJECT_ID`
- `FIREBASE_PRIVATE_KEY`
- `FIREBASE_CLIENT_EMAIL`

## Build

Build: `yarn build`
